## **프로젝트 소개**

> 운동 방법 및 관련 상품 판매 정보 공유 웹 사이트.
> 
- 코로나로인한 집콕 생활의 장기화+ 영상시청증가 = 운동량감소
이로인한 운동방법과 운동기구에 대한 정보를 제공해주는 
커뮤니티가 필요해서 기획하였습니다.
- 헬스커뮤니티를 찾는 사람에 비해 헬스 커뮤니티가 부족.

## **차별화 전략**
- 다양한 커뮤니티 활성화로 필요 운동자세 터득 가능
- 필요 상품 구매 가능.
